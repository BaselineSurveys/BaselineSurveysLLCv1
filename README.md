# Baseline Surveys Website

Static marketing site for Baseline Surveys.

## Structure

- `index.html` – main landing page
- `about.html`, `services.html`, `careers.html`, `contact.html`, `ourteam.html`, `quote.html`, `testimonials.html`, `utility.html` – additional pages
- `css/` – stylesheets
- `js/` – scripts
- `images/` – logos and photos

## How to use

1. Open this folder in a terminal.
2. Initialize git (if not already):

   git init
   git add .
   git commit -m "Initial commit"

3. Create a GitHub repo and connect it:

   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main

Then connect that repo to Netlify and use:

- Build command: (leave blank – pure static site)
- Publish directory: `.` (a single dot)
